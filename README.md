# DoubleRoulette
**Free Roulette Application! Compared with normal roulettes, this one can decide two elements at the same time. Of course, This can be used as general roulette.**

<img src="Sources/featureGraphic.png" >

### How to use
1. Add elements by pressing the plus button on the top right.
1. Select Outer or Inner, Item's name and color(color can be selected 3 patterns in color picker mode).
1. Move to roulette by pressing the play button on the top right.
1. Press START button and have fun!
1. Slide cells to delete elements.
Very easy to use, isn't it? :)
